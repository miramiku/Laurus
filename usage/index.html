Laurus :: Usage

基本的な使い方

目的ステージの変更

ステージ名の部分（右の画像では1-1 スポーツ少女アウラ（１） 第１章 初めてのリヴィエール）にマウスカーソルを合わせると赤い枠線が表示されます。 枠線が表示された状態でクリックするとステージ選択ダイアログが表示されます。

ステージ選択ダイアログ

ステージのカテゴリは、緑が閉鎖状態で赤が開放状態を表します。 それぞれ、色の部分をクリックすると状態が反転します。
ステージ名にマウスカーソルを合わせると赤い枠線が表示され、さらにクリックするとステージを選択することができます。 ステージ選択ダイアログが閉じるとスコアの計算が終わっています。

コーディネート

それぞれの部位（スロット）に対してスコアの高いものが１つ表示されています。
表示されているアイテムを所持していない場合は、アイテムのすぐわきに表示されている>にマウスカーソルを合わせると赤くなりクリックすると次にスコアの高いアイテムが表示されます。 また、行きすぎて戻る時は反対側に表示されている<にて行います。
