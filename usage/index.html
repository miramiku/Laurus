Laurus :: Usage

基本的な使い方

目的ステージの変更

ステージ名の部分（右の画像では1-1 スポーツ少女アウラ（１） 第１章 初めてのリヴィエール）にマウスカーソルを合わせると赤い枠線が表示されます。 枠線が表示された状態でクリックするとステージ選択ダイアログが表示されます。

ステージ選択ダイアログ

ステージのカテゴリは、緑が閉鎖状態で赤が開放状態を表します。 それぞれ、色の部分をクリックすると状態が反転します。
ステージ名にマウスカーソルを合わせると赤い枠線が表示され、さらにクリックするとステージを選択することができます。 ステージ選択ダイアログが閉じるとスコアの計算が終わっています。

コーディネート

それぞれの部位（スロット）に対してスコアの高いものが１つ表示されています。
表示されているアイテムを所持していない場合は、アイテムのすぐわきに表示されている>にマウスカーソルを合わせると赤くなりクリックすると次にスコアの高いアイテムが表示されます。 また、行きすぎて戻る時は反対側に表示されている<にて行います。

詳しい使い方

はじめに

Laurus（ローラス）はニキ株式会社が配信しているスマホゲーム ミラクルニキ 内のステージにおいて、高いスコアをはじきだすのに適切だと思われるアイテムのコーディネートを補助することを目的としています。 高いスコアを出す方法の詳細は秘匿されており外部から観察できる状況からスコアの算出を試みています。 そのため実際と異なる事も多いと思いますがあらかじめご了承ください。

機能概要

Laurus は大きく分けて３つの機能を現在実装しています。

  • Advisor : 最適コーディネートを求める機能
  • Wardrobe : アイテムの抽出、所持状況管理を行う機能
  • Cheatsheet : 最適コーディネート早見表の出力
全般的な使い方

それぞのれページの移動は、ページ右上にあるメニューにて移動できます。 (1) 文字の後ろに色がついているものが現在位置となります。 (2) もっとも右にある「」にマウスカーソルを合わせると、メニューが拡張されます。 (3) そして、マウスカーソルを文字の上に合わせると枠線が付きますのでクリックするとページを移動することができます。

ページ右側には目次が表示されています。 通常時は薄いですが、マウスカーソルを合わせると濃くなります。 また、文字にマウスカーソルを合わせると少し伸びます。 少し伸びている状態でクリックするとその内容の場所まで移動します。

Laurus :: Advisor

最適コーディネートを探し出す Laurus のメイン機能です。

目的条件

ブラックリスト・ホワイトリストを除いて編集することが可能です。
評価基準を編集した状態で条件確定をクリックすると編集した内容で最適コーディネートを探し出します。
シナリオのステージに関してはシナリオクラスが表示され、それぞれの文字をクリックすることで表示を切り替える事ができます。

評価基準の編集

該当するスタイルの-または数値にマウスカーソルを合わせると枠線が表示され、クリックすることで編集することができます。
タグの部分にマウスカーソルを合わせ背景に色がついている時にクリックするとタグ選択ダイアログが表示されタグを選ぶことができます。

推奨コーディネート

リセットボタンをクリックするとすべてのスロットにある推奨アイテムをすべて先頭に戻します。
例えば、右の動画内に表示されているR-ヒツジ娘は所持しておらず入手困難なアイテムなどの場合は次回から表示しないようにできます。 所持情況を変更するにはまず編集ロックをクリックして編集可能という表示にします。 編集可能の状態でアイテム（ここではR-ヒツジ娘）をクリックすることで所持情況を変更できます。 ふたたび編集可能をクリックすることで編集ロックに戻り、編集できなくなります。
ここで編集された所持情況は Laurus :: Wardrobe において不所持状態を解除することができます。

その他の機能

ワンピース対トップスとボトムスなど、択一のスロットに関しては合計スコアの高いほうに[推奨]、低い方に[非推奨]がつけられています。 また、これらは今表示中のアイテムから計算されています。 そのため、現在表示中のアイテムを変更するたびに再評価されて推奨スロットが変わることがあります。
アクセサリに関して、アクセサリの中でスコア順位が表示されています。 この順位も現在表示中のアイテムから計算されているので表示しているアイテムを変更するたびに表示が変わる可能性があります。

Laurus :: Wardrobe

アイテムの検索や抽出を行う機能です。
現在は所持情況の編集のみ雑に記載しています。 のちにきちんと記述したものに差し替えます。

所持情況の編集

並べ替えの右付近にマウスカーソルを合わせると緑色の枠線が表示されるのでクリックすると並べ替え設定ダイアログが表示されます。 並べ替え設定ダイアログの下にある未所持アイテムを表示するをクリックして色をつけると未所持のアイテムが表示されます。 また、さらに表示された未所持アイテムのみ表示をクリックすることで未所持に設定されているアイテムのみが表示されるようになります。 あとは Advisor の時と同様に編集ロックを解除して所持設定の変更が行えます。
